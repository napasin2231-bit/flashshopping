# Demo video script for TikTok review

Record a 60-90 second screen video showing this flow. Attach it to TikTok support only if they ask for more proof.

## English narration

```text
This is Flash Shopping Manager, a creator workflow tool for social commerce publishing.

First, the creator imports product data such as item name, price, affiliate link, and product folder.

Next, the app prepares product images, creates a vertical promotional video, and generates a caption.

The creator can review the video and caption before publishing.

For TikTok, the user connects their own TikTok account through OAuth. The app does not ask for or store TikTok passwords.

After the user selects the connected account and confirms the content, the app submits the prepared video and caption through the approved TikTok Content Posting API workflow.

The dashboard records each publishing attempt with platform, item, status, timestamp, and error message if any.

The public website includes the product overview, demo workflow, Privacy Policy, Terms of Service, and contact information.
```
