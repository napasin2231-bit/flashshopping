What the app does

A complete creator operations tool, not a placeholder website.

Product asset management

Keep each item in its own folder with product photos, generated preview images, final videos, captions, and short affiliate links.

Short-form video preparation

Create vertical promotional videos optimized for mobile viewing, including product title, price, call to action, and visual preview.

TikTok content posting workflow

Connected TikTok accounts can authorize posting through OAuth. The app prepares a video and caption before publishing through approved API access.

Facebook Reel workflow

Creators can also manage Facebook Page Reel status, caption reuse, and post history from the same queue.

Token and account status

The dashboard shows connected account status, token expiry warnings, failed posts, and items that still need manual attention.

Review and compliance logs

Every publish attempt is logged by platform, item, timestamp, status, and message so creators can audit their workflow.

TikTok integration

How TikTok is used

Flash Shopping Manager uses TikTok only for creator-authorized content posting. Users connect their own TikTok account, select a prepared video, review the caption, and publish through the approved TikTok Content Posting API scope.

  • OAuth-based account connection
  • No password storage
  • User-selected video and caption
  • Publishing status and error logs
  • Privacy Policy and Terms visible publicly

Reviewer access

No login is required to inspect this public website.

The product overview, demo, Privacy Policy, Terms of Service, and contact details are public. If a reviewer needs a test account for the private application dashboard, credentials can be provided in the TikTok Apply Reason field.
